Methodology · Research protocol

Pre-registered, reproducible, peer-review-ready.

Six stages. Open data. Open code.

01
DESIGN

Stratified across climate, era, HVAC class, water-damage history.

02
CONSENT

Exempt-category IRB via UT Austin. Heaven.com consumer onboarding.

03
COLLECT

HVAC filter cut-out, surface swab, blank control. Ambient mail-in.

04
EXTRACT

QIAGEN PowerSoil Pro. Blind-coded 96-plex with blanks.

05
SEQUENCE

KAPA HiFi + barcoded ITS1F/ITS4 → SMRTbell → Revio. 1,536-plex.

06
ANALYZE

DADA2 → UNITE → LamaTerra risk classifier. Cross-validated.

Statistical power
  • Cohort 3,000 properties
  • 3 samples / property
  • 9,500 samples total
  • 80% power: Δ ≥ 0.5% abundance
Controls & rigor
  • Blank + extraction blank / 96-plex
  • Mock community / SMRT Cell
  • Double-blind classifier scoring
Open science
  • Pre-registered on protocols.io
  • Data on ENA / SRA
  • Code on GitHub
  • Peer-reviewed methods paper
